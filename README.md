# 🧘 ZenForces 

ZenForces is a lightweight Chrome extension designed for competitive programmers. It transforms Codeforces into a peaceful, distraction-free environment by hiding ratings, ranks, colored usernames, and standings, helping you conquer rating anxiety and focus purely on algorithmic problem-solving.

## ✨ Features
* **Neutral Usernames:** Strips all rank colors and bolding from usernames globally.
* **Standings Hidden:** Completely removes the standings and rating change tables.
* **Graph Destroyer:** Actively blocks and hides the profile rating graph.
* **Distraction-Free Contests:** Hides the number of participants who solved a problem.
* **Gym Zen Toggle:** A native, interactive button injected directly into the Codeforces menu bar allowing you to toggle standings ON or OFF during practice Gyms.

## 🚀 How to Install
Since this extension is currently open-source and not on the Chrome Web Store, you can install it manually in seconds:

1. Click the green **Code** button at the top of this repository and select **Download ZIP**.
2. Extract the downloaded ZIP file to a folder on your computer.
3. Open Google Chrome and navigate to `chrome://extensions/`.
4. Toggle on **Developer mode** in the top right corner.
5. Click the **Load unpacked** button in the top left.
6. Select the extracted `ZenForces` folder.
7. Refresh Codeforces and enjoy the peace!
